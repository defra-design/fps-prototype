Using this Form

This form allows you to update all invoices of the type indicated.

Typically you will need a spreadsheet with data & layout identical to those shown in the datasheet.

You can then delete all existing records from the datasheet and paste in new ones from the spreadsheet.

Tip: To highlight all records click on the square in the top left hand corner of the datasheet. Then press the delete key to remove them.

If the database integrity rejects any pasted records, these will be entered in to a table called "Paste Errors". Use the button to view & print them.

NB "Paste Errors" will be overwritten with then next set of rejected records
