This form allows you to over-ride the normal WIP calculation with a figure you enter manually.

To do this, enter the amount in the WIP_Current field. You can for example enter £ 0 to default the WIP to £0

Where the field is blank the system will calculate WIP using the agreed formula from WIP_EOY etc.

The projects are listed alphabetically. If you wish to find a particular project use the "binoculars" on the tool bar.
